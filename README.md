## CryptoParsAndVisual

Скелет проекта для потока данных криптовалют: Go-сервисы для сбора и агрегации, Airflow для оркестрации, ClickHouse/PostgreSQL как витрина, Tableau/BI для визуализации. Репозиторий содержит базовую структуру директорий, примерные конфигурации и шаблоны сервисов.

### Скоростной старт
- `deploy/docker-compose.yml` разворачивает локальное окружение (`docker compose up -d`).
- Сервис `collector` опрашивает публичные REST API и складывает данные в очередь/БД.
- DAG Airflow запускает контейнер `aggregator`, агрегирует данные и сохраняет их в витрину.

### Директории
- `collector`, `aggregator`, `migrator` — исходники и Dockerfile Go-сервисов.
- `airflow` — DAG, плагины и сборка Airflow.
- `deploy` — Compose, переменные окружения, конфигурации, скрипты.
- `monitoring`, `tableau`, `docs`, `infra` — вспомогательные артефакты.
